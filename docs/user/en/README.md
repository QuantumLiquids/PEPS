# PEPS User Documentation (EN)

This is the GitHub-first user documentation for QuantumLiquids/PEPS.

## Overview

QuantumLiquids/PEPS is a high-performance C++ library for finite-size PEPS/TPS algorithms for 2D quantum many-body systems, including Simple Update, VMC optimization, and Monte Carlo measurement of observables.
