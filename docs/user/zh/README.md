## 项目概览

QuantumLiquids/PEPS 是一个用于有限尺寸 Projected Entangled Pair States (PEPS)
与相关张量网络算法的开源 C++ 库，用于高效模拟二维有限尺寸量子多体系统。

适用场景包括（不完全列表）：
- 凝聚态强关联体系：磁性与超导
- 量子计算与量子模拟平台
- 冷原子体系
